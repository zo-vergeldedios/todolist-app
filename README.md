# Task Manager App

A simple task management application that allows users to create daily tasks and assign time allocations for each task. The goal is to help users structure their day and practice basic productivity tracking.

---

## Description

This project was built using plain JavaScript, HTML, and CSS with a Node.js and Express backend.

It helped me understand how to build a dynamic UI without frameworks and how to connect a frontend structure with a basic server setup.

The app focuses on core JavaScript fundamentals such as DOM manipulation, state handling, and event-driven updates.

---

## What I Practiced

This project was designed to strengthen my understanding of:

### JavaScript Fundamentals
- DOM manipulation using `document.createElement` and `querySelector`
- Event handling (`onClick` and function binding)
- Functions and modular UI creation
- Array manipulation for storing application state
- Dynamic rendering using a custom `render()` function

### Frontend Architecture (Vanilla JS Component Thinking)
- Building reusable UI elements using a custom `element()` function
- Simulating component-based architecture without frameworks
- Re-rendering UI by replacing DOM nodes (`replaceChildren`)

### Backend Basics (Node.js + Express)
- Setting up a basic Express server
- Understanding server initialization and port listening
- Preparing for API structure (GET/POST concepts planned but not fully implemented)

### Data Handling
- Managing in-memory state using JavaScript objects
- Structuring task and time data in a simple state object
- Preparing for future persistence layer integration

---

## Features

- Add tasks through an input field
- Assign a time value to each task
- Dynamic UI rendering without page refresh
- Basic state management using JavaScript objects
- Modular UI creation using a helper function (`element()`)
