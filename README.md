# To Do List App:

This To Do List App can help you list out the tasks for the day, you can also indicate the time you would like to allot for each task.

# Description

This app was created using plain JavaScript, CSS and HTML.
It helped me practice and learn about DOM, functions, setTimeout, array methods etc.

# Instructions

1. Clone the repository to your terminal.
2. Right click index.html and click 'Open with Live Server'
